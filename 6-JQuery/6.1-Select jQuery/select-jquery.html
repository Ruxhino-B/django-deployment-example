Selecting with jQuery

You can use the $ sign to select elemt from the DOM! It will feel a lot like selecting using querySelectorAll().

For Ecample:

  • Selecting All a tags:
    • $('a')
  • Selecting all elements from the class "container"
      li$(".container")
  • Selecting all element with id "special"
    • $("#special")
