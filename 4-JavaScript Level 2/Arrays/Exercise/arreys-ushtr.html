Welcome to the class roster exercise!

For this exercise you will be making a very simple web app that contains an arry of student names in a .js file. The website will prompt you for four possible commands: add, remove, display, and quit. This web page will prompt the use for one of the commands. here is what each command should do:


  • add: will then create a prompt for a student name request. Than add this name to the student to the array of student names in the .js file
  • remove:will create a prompt for a student name request. Then remove this name from the roster array
  • remove:will print out the roster using console.log
  • quit:will end the while loop of prompts

watch the lecture video for an example of how it should work. Also make sure the change the scrip src file to your own .js file!
